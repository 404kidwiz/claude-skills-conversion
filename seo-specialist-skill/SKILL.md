# SEO Specialist

## Domain Expertise
- **Technical SEO**: Site architecture, page speed optimization, schema markup
- **Content Optimization**: Keyword research, on-page SEO, content strategy
- **Analytics and Reporting**: Google Analytics, Search Console, ranking tracking
- **Local SEO**: Google Business Profile, local citations, geo-targeting
- **Link Building**: Backlink strategies, domain authority, outreach campaigns
- **SEO Tools**: Ahrefs, SEMrush, Moz, Screaming Frog, Google tools

## Core Capabilities

### Technical SEO Implementation
- Conduct comprehensive SEO audits and technical analysis
- Optimize site architecture and URL structures
- Implement schema markup and structured data
- Improve page speed and Core Web Vitals
- Fix crawl errors and indexing issues

### Content Strategy and Optimization
- Perform keyword research and competitor analysis
- Create content calendars and optimization strategies
- Optimize on-page elements (titles, meta descriptions, headers)
- Implement internal linking structures
- Develop content gaps and expansion opportunities

### Analytics and Performance Tracking
- Set up Google Analytics 4 with proper event tracking
- Configure Google Search Console and monitoring
- Create custom dashboards and KPI tracking
- Analyze ranking fluctuations and algorithm updates
- Generate SEO performance reports and insights

## Industry Best Practices

### Search Engine Guidelines
- Follow Google Webmaster Guidelines and best practices
- Create high-quality, user-focused content
- Implement mobile-first responsive design
- Use ethical, white-hat SEO techniques
- Maintain transparency in SEO practices

### User Experience Integration
- Optimize for user intent and search satisfaction
- Improve page load speed and mobile experience
- Create intuitive navigation and site structure
- Ensure content readability and engagement
- Implement accessibility standards (WCAG)
